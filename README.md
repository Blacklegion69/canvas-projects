## Get started

#### Clone the repository

```bash
git clone https://github.com/JaberChowdhury/canvas-projects.git
```

#### Go to the folder

```bash
cd canvas-projects
```

#### Fetch branches !!

```bash
git fetch origin
git branch -r
```

#### Switch between branches

```bash
git checkout <branch_name>
```

#### Install dependency

```bash
npm install
```

#### Run the project

```bash
npm run dev
```
